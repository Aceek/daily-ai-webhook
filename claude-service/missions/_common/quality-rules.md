# Quality Rules (Common)

Ces règles s'appliquent à TOUTES les missions.

## Critères de qualité obligatoires

### Sources
- Toujours privilégier les sources primaires (blog officiel, communiqué)
- Croiser les informations avec au moins une recherche web
- Ne jamais inclure de rumeurs non confirmées

### Contenu
- Titres factuels, pas de clickbait
- Résumés de 2-3 phrases maximum
- Focus sur le "quoi" et le "pourquoi", pas le "comment"
- Pas d'opinions personnelles

### Vérification
- Sources officielles = confidence "high"
- Sources réputées non vérifiées = confidence "medium"
- Sources douteuses = fact-check obligatoire avant inclusion
- Confidence "low" = NE PAS inclure dans le digest

### Exclusions systématiques
- Contenu promotionnel sans substance
- Articles de plus de 48h (sauf événement majeur en cours)
- Doublons (même news, sources différentes)
- Rumeurs et spéculations non sourcées

## Processus de décision

```
Source fiable ?
    ├─ OUI → confidence: high → INCLURE
    └─ NON → fact-check
                ├─ VERIFIED → confidence: medium → INCLURE
                └─ NOT VERIFIED → EXCLURE + noter dans research doc
```
